Why AWS

Startup Ecosystem

AWS Activate gives early-stage startups $5K–$25K in free credits. As a Cornell-backed team, Redtail likely qualifies for the Founders tier — effectively making our entire server bill $0 for Year 1.

Python-Native Tooling

All our scrapers, Claude integrations, and LangChain/LlamaIndex agents are Python. AWS Lambda + EC2 have the deepest Python support, the most mature boto3 SDK, and the largest community of examples.

Scale-When-Ready

Start with a $35/mo t3.medium. When Game 3 ships, auto-scale to multiple instances without changing any code. No infrastructure rewrite as we grow from prototype to production.

Managed Services = Less Ops

Lambda handles cron jobs (no server to maintain). SQS handles job queuing. CloudWatch handles monitoring. A 4-person team shouldn't be managing DevOps — AWS handles it for us.

Cheapest Compute Per Dollar

EC2 Spot Instances for non-critical scraping jobs cut costs by 70–90%. S3 at $0.023/GB is the cheapest durable storage available. Lambda is free for first 1M requests/month.

Future-Proof for AI/ML

When we're ready for custom model training (Year 2), SageMaker is on the same platform. No migration needed. Bedrock also offers Claude API access directly through AWS billing.

Service-by-Service Breakdown
AWS Service What We Use It For Phase Monthly Cost
EC2 (Elastic Compute)
t3.medium → t3.large → t3.xlarge
 The main workhorse. Runs 24/7 Python scraper agents, Crew AI orchestrator, LangChain agent pipeline, and Supabase connection pooler. Everything that needs to always be "on" lives here. Phase 1
$35 → $70 → $140
Scales with number of active titles
Lambda
Scheduled cron triggers
 Fires scraping jobs every 6 hours, triggers analysis pipelines when new data lands in S3. No server to maintain — just a function that wakes up on a schedule. First 1M requests/month are free. Phase 1
~$5/mo
First 1M requests free forever
S3
Raw data archive & staging
 Stores all raw scraped data (Reddit dumps, YouTube comment exports, Steam review exports) as JSON or Parquet files before they're cleaned and loaded into Supabase. Acts as a data lake — cheap, durable, queryable with Athena later. Phase 1
~$3–8/mo
$0.023/GB · first 5GB free
SQS (Simple Queue Service)
Job queue for async agents
 When Lambda fires a scrape trigger, it drops a job into SQS. EC2 scraper agents pick up jobs from the queue. Prevents agents from stepping on each other, handles retries automatically. First 1M requests/month free. Phase 1
~$0–2/mo
First 1M requests free forever
CloudWatch
Monitoring, logs, alerts
 Collects logs from EC2 scrapers and Lambda functions. Alerts us if a scraper fails or an agent crashes. Dashboards for API usage, error rates, and data pipeline health. Mostly within the free tier for our scale. Phase 1
~$0–3/mo
Largely within free tier
IAM + Secrets Manager
Security & API key management
 All API keys (Claude, Sensortower, Apify, Twitter/X) stored in Secrets Manager — never hardcoded. IAM roles control exactly what each Lambda and EC2 instance can access. Non-negotiable security baseline. Phase 1
~$2/mo
$0.40/secret/month · 5 secrets
RDS / Supabase
Primary signal database
 We use Supabase (managed Postgres) rather than native RDS to save costs early. Supabase free tier handles M1–M3. Upgrade to $25/mo Pro plan at M4 when data volume and concurrent agent queries increase. Phase 2
$0 → $25/mo
Free tier covers first 3 months
ElastiCache / Upstash Redis
Agent response caching
 Claude API calls are expensive. When an agent has already synthesized "what do Reddit users say about idle game monetization," we cache the result for 24 hours. This alone can reduce Claude API costs by 20–40% at scale. Phase 2
$0 → $10/mo
Upstash serverless Redis
EC2 Auto Scaling / ECS Fargate
Scale compute with titles
 Phase 3 migration: move from a single EC2 instance to containerized agents on ECS Fargate. Each scraper agent becomes its own container — spin up 10 in parallel during heavy scraping, scale back to 1 overnight. Pay only for what you use. Phase 3
~$100–200/mo
Replaces single EC2 at scale

AWS vs Alternatives

✓ AWS — Recommended

  • $5K–$25K free credits via Activate
  • Deepest Python / boto3 ecosystem
  • Lambda + SQS + S3 — perfect for agent workloads
  • SageMaker ready for Year 2 ML
  • Best documentation + Stack Overflow coverage
  • Bedrock = Claude API through AWS billing
  • Slightly more complex to set up than GCP

GCP — Strong but Second

  • Google Cloud for Startups: $200K in credits (larger but harder to qualify)
  • Vertex AI + BigQuery are excellent for ML at scale
  • Cloud Run is cleaner than Lambda for Python
  • Firebase already in our stack (adds complexity)
  • Smaller community for our specific agent use case
  • Gemini API = competing with our Claude dependency

DigitalOcean — Not Yet

  • Simplest interface — good for a single developer
  • Cheapest raw compute ($6/mo droplet)
  • Good managed Postgres option
  • No serverless functions equivalent to Lambda
  • No managed queue service equivalent to SQS
  • Limited AI/ML managed services
  • Would need to rebuild entire stack if we outgrow it
