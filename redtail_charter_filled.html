Contextual note
Key notes:
1. Total infrastructure cost ($13,147/year) = ~11% of the $120K pre-seed — highly capital-efficient. Developer/CTO salaries and MKT/ADS are budgeted separately per the Use of Funds slide.
2. Months 1–3 run lean at ~$350–400/mo — using free tiers for Supabase, Redis, Firebase. Only core tools active.
3. Sensortower ($500/mo) is the largest single line item. Strategy: defer until Month 4 when Game 1 launches and revenue begins. First-party Firebase data covers M1–M3 analytics needs.
4. AWS Activate Program: Apply immediately. As an early-stage startup, Redtail likely qualifies for $5,000–$25,000 in free AWS credits — effectively making the Servers section $0 for the first 12 months.
5. Crew AI, LlamaIndex, LangChain: All open source. Run on the same EC2 instance as scrapers. Zero additional cost.
6. Firebase + Game Analytics: Free tiers cover all post-launch analytics needs (D1/D7/D30, ARPU, sessions) until the user base exceeds ~50K MAU.
